\course{CS404. Seminario de Tesis}{Obligatorio}{CS404}

\begin{justification}
Este curso tiene por objetivo que el alumno entregue su borrador de tesis.
\end{justification}

\begin{goals}
\item Que el alumno complete este curso con su tesis lista para una posible sustentación inmediata.
\item Que el alumno presente formalmente el borrador de tesis ante las autoridades de la facultad.
\item Los entregables de este curso son:
	\begin{description}
	\item [Avance parcial:] Avance del proyecto de tesis incluyendo introducción, marco teorico, estado del arte, experimentos, y bibliografía sólida.
	\item [Final:] Proyecto de tesis completo y listo para sustentar en un plazo no mayor de quince días.
	\end{description}
\end{goals}
